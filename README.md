# Smart Home Automation System using ESP32

## Overview

This project is a Bluetooth-based Smart Home Automation System developed using ESP32 and relay modules. The system allows users to control electrical appliances through voice commands using an Android application.

## Features

* Bluetooth-based communication
* Voice-controlled appliance operation
* ESP32 microcontroller
* Relay interfacing for device control
* User-friendly Android application
* Low-cost home automation solution

## Components Used

* ESP32 Development Board
* Relay Module
* Android Smartphone
* Bluetooth Module (if used separately)
* Electrical Appliances

## Technologies Used

* Embedded C / Arduino C++
* ESP32
* Arduino IDE
* Android Application Development
* Bluetooth Communication

## Working

The Android application converts voice commands into control signals and sends them to the ESP32 via Bluetooth. The ESP32 processes the commands and switches connected appliances ON or OFF through relay modules.
